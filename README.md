# Multimodal Alzheimer's Disease Detection from Speech

This project investigates machine learning methods for detecting Alzheimer's disease from speech using the DementiaBank Pitt Corpus Cookie Theft picture-description task.

The goal is to compare acoustic, linguistic, semantic, and multimodal fusion approaches for session-level Alzheimer's disease classification.

## Project Overview

Alzheimer's disease can affect memory, executive function, language, fluency, and speech production. The Cookie Theft task provides both spoken audio and transcripts, making it useful for comparing speech-based and language-based machine learning models.

This project evaluates:

- Acoustic features from speech recordings
- Text-based features from transcripts
- Semantic features based on Cookie Theft image content
- Late fusion between text, audio, metadata, and semantic models

## Dataset

This project uses the DementiaBank Pitt Corpus.

Due to data-use restrictions, the dataset is not included in this repository. One must request access through DementiaBank/TalkBank and replace the zip file path before running the pipeline.

## Methods

### Text Models

- TF-IDF 1–2 gram features
- Logistic regression baseline
- RoBERTa-based transcript classification
- Sentence-transformer embeddings using `sentence-transformers/all-MiniLM-L6-v2`

### Acoustic Models

- MFCC features
- Log-mel spectrogram features
- HuBERT embeddings
- Layer-level HuBERT feature analysis

### Semantic Models

The semantic branch compares transcript content against expected Cookie Theft image concepts generated by CHATGPT LLM. This is used to measure whether the participant mentions important objects, people, and events in the image.

### Fusion

Late fusion is used to combine model probabilities from different branches. Fusion weights and decision thresholds are selected on the development set and evaluated on the held-out test set.

## Results

The best multimodal fusion model achieved:

| Model | Accuracy |
|---|---:|
| Text baseline | 83.8% |
| Multimodal fusion | 86.8% |

The results suggest that linguistic features were stronger than acoustic features for this task, while semantic/image-grounded features provided additional useful information.

## Key Finding

Text-based models performed strongest overall, but multimodal fusion improved performance by combining complementary information from transcript and semantic features.
